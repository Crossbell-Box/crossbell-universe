# `useIsWalletSignedIn`

Hook for indicating whether a wallet is signed in. If you are checking whether if Operator Sign is enabled, use [useIsOpSignEnabled](docs/hooks/use-is-op-sign-enabled) instead.

```ts
function useIsWalletSignedIn(): boolean;
```

## Usage

```tsx {2,7,15}
import {
	useIsWalletSignedIn,
	useOpSignSettingsModal,
} from "@crossbell/connect-kit";

function App() {
	const isWalletSignedIn = useIsWalletSignedIn();
	const opSignSettingsModal = useOpSignSettingsModal();

	return (
		<button
			onClick={() => opSignSettingsModal.show()}
			disabled={isWalletSignedIn}
		>
			{isWalletSignedIn ? "Signed In" : "Sign In"}
		</button>
	);
}
```
